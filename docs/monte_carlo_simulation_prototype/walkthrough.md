# 完了報告: モンテカルロ・シミュレーション プロトタイプ

競艇の3連単予測に向けたモンテカルロ・シミュレーションのプロトタイプを実装しました。

## 実施内容

### 1. シミュレーションエンジンの作成
- `monte_carlo_simulation.py` を新規作成しました。
- NumPyのベクトル演算を活用し、1万回分のレースを高速に一括シミュレートします。

### 2. 展開ロジックの実装
- スタートタイミング（ST）の優劣が着順に与える影響を組み込みました。
- 簡易的な「4カド捲り」ロジックを実装し、スリット隊形による勝率の変動をシミュレートしています。

### 3. 期待値によるフィルタリング
- シミュレーションから得られた「独自の出現確率」と「仮想オッズ」を掛け合わせ、期待値を算出します。
- ユーザーが引数 `threshold` を調整することで、期待値の高い（エッジのある）買い目のみを抽出できます。

## 検証結果

スクリプトの実行により、以下の通り期待値閾値に応じてベット点数（購入機会）が変化することを確認しました。

- **閾値 1.0以上**: 27点（利益が出る可能性のある買い目すべて）
- **閾値 1.1以上**: 20点（中程度の自信）
- **閾値 1.5以上**: 6点（非常に高いエッジを持つ買い目のみ）

これにより、回収率を重視しつつも、楽しむために購入頻度を調整したいというユーザーのニーズに応える土台が整いました。
