# ボートレース予測・投資戦略エンジン V3.1+ 開発完了報告書

## 実施内容
ボートレース予測モデル V3.1 を基盤とし、的中確率を利益に変換するための「資金配分・期待値最適化エンジン」を実装しました。これにより、AI の予測値を「どの買い目に、いくら投資すべきか」という具体的な投資戦略へ昇華させました。

## 新機能：投資戦略モジュール
1. **全120通り確率計算**:
    - ハルビルの公式 (Harville's formula) を実装し、スタッキングモデルの各着順確率から 3連単全組み合わせの的中確率を精密に算出。
2. **期待値 (EV) フィルタリング**:
    - `期待値 = 的中確率 × オッズ` をリアルタイム計算。ユーザーが指定した閾値（例: 1.5以上）を超える「利益の出る買い目」のみを自動抽出。
3. **2大資金配分エンジンの搭載**:
    - **ケリー基準 (Kelly Criterion)**: 理論上、破産リスクを回避しつつ複利利益を最大化する最適な投資比率を算出。
    - **均等払い戻し (Dutching)**: どの買い目が的中しても同等の利益（または払い戻し）が得られるよう、投資額を自動分配。
4. **100円単位の具体的指示**:
    - 入力された総予算に対し、端数を切り捨てた実戦的な投資金額を提示。
5. **バックテスト・シミュレーター**:
    - 過去データに対し、「均等買い」「フルケリー」「ハーフケリー」などの戦略を適用した際の収支曲線を比較可視化。

## 検証結果
- **モデル精度 (V3.1)**: LogLoss 0.3478 / Accuracy 86.22%
- **戦略シミュレーション**: ハーフケリー基準 (Coef=0.5) を用いることで、ボラティリティを抑えつつ期待値を追う投資行動が可能であることを確認。

## ファイル構成
- `app_v3.py`: 投資戦略エンジンを統合した最新アプリケーション
- `train_v3.py`: V3.1 学習スクリプト
- `models/`: 統合された各モデル・スケーラーファイル

## 次のステップ
以下のコマンドで、進化した「投資プラットフォーム」を起動してください。
```bash
streamlit run boatrace-v3-predictor/app_v3.py
```
メニューから「期待値閾値」や「戦略」を切り替えることで、瞬時に最適な買い目が更新されます。
