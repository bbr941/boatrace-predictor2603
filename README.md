# BoatRace AI V3.1+ (Investment Strategy Engine)

ボートレース（競艇）の的中確率予測と、期待値に基づく投資戦略を提示するAIシステムです。

## 特徴
- **アンサンブル推論**: LightGBM, CatBoost, および高度な正則化を施したメタモデルによるスタッキング構成。
- **リアルタイム解析**: ボートレース公式サイトから展示情報を自動スクレイピングし、最新のオッズを反映。
- **投資戦略エンジン**: ハルビルの公式による3連単確率計算、ケリー基準による最適な資金配分。
- **透明性**: デバッグモードにより、AIが使用した生データや抽出プロセスを可視化。

## セットアップ

### ローカル実行
```bash
pip install -r requirements.txt
streamlit run boatrace-v3-predictor/app_v3.py
```
